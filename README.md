 실시간채팅을 할수 있는 간단한 실시간 채팅앱을 제작하게 되었습니다.

### 🍾역할 
-  erd 설계  
-  RestTemplate으로 외부의 API를 사용하여 데이터 수집  
-  상품 상세 페이지 및 카테고리 구현   
-  AJAX를 사용한 결제 및 취소 요청 구현   
-  AWS서버에 배포를위한 자동화 스크립트 작성     
-  기능 통합시 발생한 에러 해결
